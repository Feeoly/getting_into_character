<threat_model>
## Trust Boundaries

| Boundary | Description |
|---------|-------------|
| 用户文件输入 → 本地存储 | 上传图片从 File/Blob 进入 IndexedDB（需要类型校验与大小/类型约束） |
| 本地资源渲染 → 浏览器媒体子系统 | `<video>` 背景与后续录制会触达媒体 API（需要权限与错误可见性） |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|----------|----------|-----------|-------------|-----------------|
| T-02-01 | Information Disclosure | 背景资源加载 | mitigate | 预置背景全部放 `public/` 本地打包；上传背景仅用 Blob/object URL，不允许外站 URL；切换时 revoke object URL 防泄漏 |
| T-02-02 | Tampering | rehearsal settings 存储 | mitigate | 使用 Zod 校验并对阈值/枚举做范围限制；读取失败回退到默认值（仍满足 D-08） |
| T-02-03 | Denial of Service | 上传超大图片导致存储/内存压力 | mitigate | 在保存上传背景前校验 `mimeType` 为 image/* 且大小限制（例如 ≤ 10MB）；失败时给出可理解错误文案 |
</threat_model>
